# Todolist-MK2

### features

- [x] Global state management with `Context API`
- [x] Reusable Components
- [x] Barrel export pattern
- [ ] tailwind with variants
  - [ ] `clsx`
  - [ ] `twMerge`
- [ ] Schema validation with `Zod` and `React-Hook-Form`
